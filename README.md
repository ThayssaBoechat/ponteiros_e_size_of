# Ponteiros e sizeof em C

Programa didático desenvolvido em linguagem C com o objetivo de demonstrar
o uso de ponteiros e do operador sizeof.

## Objetivo
Compreender como funcionam os ponteiros, o acesso indireto a variáveis
e o tamanho em memória de tipos básicos e ponteiros.

## Conteúdos abordados
- Declaração de variáveis
- Ponteiros
- Operador sizeof
- Acesso indireto a valores
- Manipulação de dados via ponteiros

## Observação
Este código foi desenvolvido para fins de estudo e aprendizado
dos conceitos iniciais da linguagem C.
